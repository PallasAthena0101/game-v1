# 小偷还我外卖

Three.js 3D 跑酷版本。

## 运行

```bash
python -m http.server 8123 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:8123/
```

## 操作

- `A` / `←`：左移
- `D` / `→`：右移
- `W` / `↑`：跳跃
- `S` / `↓`：滑铲
- `Space`：怒气爆发
- `Enter`：开始 / 重开
- `P` / `Esc`：暂停

## 机制

- 3D 追逐镜头和三车道空间。
- 跳跃、滑铲、变道分别处理不同障碍。
- 贴脸闪避触发慢动作、额外分数和怒气。
- 随机任务会驱动每局目标，完成后给高额奖励。
- 餐盒完整度替代传统血条，撞击会降低完整度。
- 怒气满后可短时间加速、无敌并扩大冲分窗口。
